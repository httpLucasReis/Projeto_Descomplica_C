Breve História da Linguagem C

Por: Eduardo Casavella

A linguagem C foi criada em 1972 e implementada em um computador DEC PDP-11 por Dennis Ritchie no Bell Laboratories . C é derivada de duas outras linguagens: Algol 68 e BCPL.

O foco da linguagem C inicialmente foi o desenvolvimento de sistemas operacionais e compiladores. C foi usada com grande êxito na construção de uma nova versão do sistema operacional Unix, que inicialmente foi escrito em Assembly. O grande sucesso obtido no mundo do Unix fez com que a linguagem ganhasse mais e mais adeptos e atualmente, quase todos os grandes sistemas operacionais são construídos em C/C++.

No ano de 1978 foi publicado o livro: The C Programming Language por Kernigham & Ritchie. Este livro fez grande sucesso e ajudou muito a divulgar a linguagem. A padronização da linguagem C

Um grande esforço de padronização foi feito a fim de padronizar a linguagem. Atualmente ainda há versões de compiladores próprias de cada fabricante, porém a maioria dos fabricantes oferece uma opção de compatibilidade com o padrão ANSI C.

A linguagem C é uma linguagem de propósito geral, o que quer dizer que se adapta a praticamente qualquer tipo de projeto, altamente portável e extremamente rápida em tempo de execução. A linguagem C++ é uma evolução da linguagem C que incorpora orientação a objetos. Linguagens como Java e C# foram influenciadas pela linguagem C. Até a próxima!
